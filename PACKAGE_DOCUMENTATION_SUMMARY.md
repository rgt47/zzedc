- **Package Name**: zzedc

- **Type**: R Package

- **Dependencies**: 36 R packages including shiny, bslib, RSQLite, and specialized packages

### Architecture Components
- **Frontend**: Modern Bootstrap 5 interface via bslib
- **Backend**: R Shiny server with modular architecture
- **Database**: SQLite with connection pooling and extension support
- **Authentication**: Database-backed with role-based access control
- **Configuration**: YAML-based hierarchical configuration management
