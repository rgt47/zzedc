# Open-Source EDC / Clinical Trial Data Capture Resources

## Platforms

### OpenEDC
- Browser-based EDC system using CDISC ODM XML.
- Supports metadata-driven form definitions, offline capture, encrypted storage, conditional logic.
- GitHub: https://github.com/imi-muenster/OpenEDC

### LibreClinica
- Successor of OpenClinica; full EDC/CDM platform.
- Java + PostgreSQL backend.
- GitHub: https://github.com/reliatec-gmbh/LibreClinica

### clinicedc (Django-based)
- Python/Django framework for multisite longitudinal clinical trials.
- Highly customizable eSource + EDC.
- GitHub: https://github.com/clinicedc/edc

### sdtm.oak
- R-based pipeline for transforming EDC exports into CDISC SDTM datasets.
- GitHub: https://github.com/pharmaverse/sdtm.oak
